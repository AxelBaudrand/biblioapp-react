# BiblioApp React

Nombre del estudiante: Axel Baudrand

BiblioApp es una SPA creada con React y Vite para una biblioteca comunitaria. Muestra un catalogo de libros, permite filtrar por estado, buscar por titulo y destacar las novedades del catalogo.

## Ejecucion

```bash
npm install
npm run dev
```

## Uso de IA

Se uso GitHub Copilot o IA como apoyo para organizar el desarrollo por etapas, crear componentes, validar buenas practicas, preparar estilos y redactar la documentacion final. No se agregaron credenciales ni datos sensibles.
